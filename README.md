# 📋 Cadastro de Pessoas em Python

Projeto simples desenvolvido para praticar lógica de programação em Python.

## 🚀 Funcionalidades
- Cadastro de pessoas
- Validação de idade (maior de 18 anos)
- Listagem de cadastros
- Menu interativo no terminal

## 🧠 Tecnologias utilizadas
- Python

## 📌 Objetivo
Este projeto foi criado com o objetivo de treinar:
- Estruturas de repetição (while)
- Condições (if, elif, else)
- Listas
- Dicionários
- Validação de dados com try/except

## ▶️ Como executar
1. Abra o terminal
2. Execute o arquivo:

```bash
python cadastros.py
